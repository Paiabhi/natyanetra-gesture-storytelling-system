# NatyaNetra - Gesture Based Storytelling System

## Overview

NatyaNetra is an adaptive embedded system that recognizes Bharatanatyam mudras using computer vision and triggers interactive storytelling outputs through an MSPM0 microcontroller.

## Features

- Real-time hand tracking
- Bharatanatyam mudra recognition
- MediaPipe-based landmark detection
- UART communication
- Embedded sensory feedback
- Interactive storytelling framework

## Technologies Used

- Python
- MediaPipe
- OpenCV
- MSPM0G3507
- UART Communication
- Embedded C

## System Architecture

Camera → MediaPipe → Gesture Classification → UART → MSPM0 → LED/Buzzer Output

## Future Enhancements

- Deep Learning based gesture recognition
- Mobile application integration
- Cloud-based story generation
- Voice narration support
